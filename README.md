# Fashion Shop - E-Commerce UI

A simple, responsive e-commerce frontend interface built entirely with pure HTML and CSS. This project does not use any external JavaScript files or complex frameworks, making it extremely lightweight and easy to run locally.

## Features

* **Pure HTML & CSS:** No JavaScript frameworks or external `.js` files are required.
* **Responsive Design:** Works well on different screen sizes.
* **Multiple Categories:** Includes separate pages for various clothing categories (Men's, Women's, Kids', Shoes, etc.).
* **Help Center:** A dedicated help and support page.
* **Local Execution:** Can be run directly from your file system without needing a web server.

## How to Run

1. Download or extract all the project files into a single folder.
2. Create a folder named `image` in the same directory and place your product images inside it (e.g., `image/Boysjens.jpg`).
3. Simply double-click the `index.html` file to open it in your default web browser.
4. Navigate through the shop by clicking on the categories and links!

## File Structure

* `index.html` - The main homepage displaying all categories.
* `style.css` - The global stylesheet containing all the design rules.
* `help.html` - The Help Center page.
* `*.html` - Various category pages (e.g., `Boys-jens.html`, `Ladies-Top.html`, `Running-Shoes.html`).
* `image/` - (You need to create this) The folder where all product images should be stored.

## Technologies Used

* HTML5
* CSS3
* FontAwesome (for icons via CDN)
